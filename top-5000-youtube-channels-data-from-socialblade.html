Introdução¶

Este relatrio faz uma anlise da base de dados Top 5000 Youtube channels data from Socialblade (https://www.kaggle.com/mdhrumil/top-5000-youtube-channels-data-from-socialblade), que uma base com informaes sobre os canais mais populares do Youtube. A finalidade deste relatrio dizer quais dos atributos numricos desta base so os mais importantes para determinar o atributo categrico Grade.

Estudando os atributos da base por algoritmos de aprendizado de máquina¶

Foi feita uma rvore de deciso para estimar os atributos mais importantes na determinao do atributo Grade. A rvore foi feita com todos os elementos da base de dados e sua finalidade no foi de predio e sim de estudar o diagrama da rvore gerada, porque o algoritmo que cria rvores de deciso tende a colocar os atributos mais importantes para a predio do atributo desejado (neste caso o Grade) mais prximos do topo do diagrama.

Diagrama truncado da rvore de deciso:

Observando o digrama nota-se que o atributo Video Views e o Subscribers so os mais importantes, isto porque eles so os atributos que mais aparecem nas caixas (ns da rvore) que esto mais altas no digrama.

Pode-se observar tambm que quanto maior o Video Views e o Subscribers melhor o Grade (assumindo que A++ o melhor Grade e B+ o pior). Para entender isto necessrio entender como se l o diagrama de uma rvore de deciso, a primeira linha de cada n da rvore uma pergunta caso seja respondida como sim deve-se seguir a seta a esquerda e caso o contrario segue-se a direita, o algoritmo da arvore de deciso faz isto ate achar um n que no possui mais perguntas (no digrama apresentado apenas o no verde no possui perguntas), caso voc no queira ir mais adiante e deseje parar em um n que possui perguntas voc ainda pode obter uma predio menos acurada que dada pela ultima linha dentro do n. Pode-se notar ento que caso seja respondido sim para as perguntas de menor ou igual para Video Views e Subscribers quase sempre alcanado um n cuja predio vai ser Grade B+.

Foi realizado um teste com grupos de modelos de predio diferentes, utilizando validao cruzada para a mdia e desvio padro da acurcia, em cada grupo de modelos predio mudado ou algoritmo de gerao do modelo e/ou o conjunto de atributos fornecidos como entrada. A hiptese que atributos, ou conjuntos de atributos, que criam modelos de predio para o atributo Grade com pouca acurcia no so importantes na determinao do Grade. Foram testados os modelos predio gerados por redes neurais, KNN e rvore de deciso. Os conjuntos de atributos utilizados foram:

import numpy as np
from sklearn.preprocessing import StandardScaler
from sklearn.model_selection import KFold
from sklearn.neural_network import MLPClassifier
from sklearn import tree
from sklearn import neighbors

def stdScaler(data):
    avg=data.mean()
    std=data.std()
    return data.apply(lambda x: (avg-x)/std )

def applyMachineLearning(clfFactory,X,Y):
    #clfFactory             => Function to create classifier object
    #X                      => Elements attributes  in dataset
    #Y                      => Elements classes in dataset
    kf = KFold(n_splits=10,shuffle=True,random_state=3223)
    acuracy=[]
    for train_index, test_index in kf.split(X):
        X_train,X_test,  = X[train_index], X[test_index]
        Y_train,Y_test  = Y[train_index], Y[test_index]
        clf=clfFactory()
        clf=clf.fit(X_train,Y_train)
        acuracy.append(clf.score(X_test,Y_test))
    return acuracy


#apply z-socre to normalization of atributes================================
df['Subscribers StdSca']=stdScaler(df["Subscribers"])
df['Video views StdSca']=stdScaler(df["Video views"])
df['Video Uploads StdSca']=stdScaler(df["Video Uploads"])

                            {"name":'KNN',"algorithm":knnFactory},

Conclusão sobre a análise exploratória¶

Este trabalho estudou como os atributos numricos da base Top 5000 Youtube channels data from Socialblade influenciam o atributo categrico Grade. O atributo Video views o mais importante para determinar o Grade, seguido pelo Subscribers, isto foi observado pelos dados obtidos na Subseo Estudando os atributos da base por algoritmos de aprendizado de mquina. A anlise da rvore de deciso gerada na mesma subseo mostra que quanto maiores estes atributos melhor o Grade, assumindo que A++ o melhor Grade e B+ o pior Grade.
