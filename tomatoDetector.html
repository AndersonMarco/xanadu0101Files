Uso de redes neurais para a identificação de frutas parecidas¶

Introdução¶

Este trabalho propõe uma análise sobre o uso de redes neurais para classificar frutas que são parecidas visualmente mas que são de diferentes tipos, criando assim um classificador de frutas. Para a realização desta tarefa foi utilizado a base de dados Fruits 360, que é uma base de dados que possui fotos de frutas, esta base esta disponível em: https://www.kaggle.com/moltean/fruits

Este trabalho apresenta uma pequena descrição sobre as frutas utilizadas, uma análise sobre os histogramas de cor para as fotos das frutas utilizadas, uma pequena descrição do tipo de rede neural utilizada para criar o classificador, códigos utilizados para a criação do classificador e uma análise sobre a eficiência que o classificador criado possui.

Os códigos e artefatos utilizados para a realização deste trabalho podem ser obtidos em: https://rebrand.ly/7sv33x

Elementos da base de dados utilizados para a análise¶

As frutas utilizadas para esta análise são de três diferentes tipos de tomate e para classificar de que tipo é cada tomate foram utilizados dados extraídos apenas de suas fotos. Um detalhe importante sobre a base de dados utilizada é que ela pode ter até três fotos de um mesmo tomate com cada foto sendo tirada de um ângulo diferente, com estes ângulos sendo fixos. As fotos utilizadas para esta análise possuem 100 pixels de comprimento por 100 pixels de altura.

def computeHistogram(imgChannel):
    hist=np.zeros(256)
    for x in imgChannel:
        hist[x]=hist[x]+1.0

    return hist

def computeAvg(histsList):
    histAvg=np.zeros(256)
    for i in range(256):
        histAvg[i]=np.mean(histsList[:,i])

    return histAvg

def computeSTD(histsList):
    histSTD=np.zeros(256)
    for i in range(256):
        histSTD[i]=np.std(histsList[:,i])

    return histSTD

Rede neural utilizada¶

Para a criação e utilização do classificador criado não são considerados o angulo da foto tirada e nem se um mesmo tomate possui varias fotos, apenas é fornecido uma foto qualquer e o classificador infere o tipo de tomate que esta na foto.

A rede neural utilizada para criar o classificador é do tipo convulacional, redes neurais deste tipo são parecidas com as as MLP (Multilayer Perceptron) possuindo também várias camadas de neurônios com a diferença de que algumas destas camadas são do tipo convulacionais. Camadas convulacionais são camadas que criam filtros para os dados deste modo, as imagens passadas para uma rede neural deste tipo não precisam de um pré-processamento, como estes filtros são criados a medida que a rede neural é treinada eles tentam ajustar-se para a forma que melhor atendam o classificador criado pela rede neural. A rede neural utiliza neste trabalho possui 4 camadas convulacionais perto da entrada e quatro camadas intermediarias do tipo MLP (ou camadas do tipo clássicas).

from keras.models import load_model

def createModel(height,width,channels,nclass):
    inputShape = (height, width, channels)

    model = Sequential()
    model.add(Conv2D(6, (5, 5), padding="same",input_shape=inputShape))
    model.add(Activation("relu"))
    model.add(MaxPooling2D((2,2)))
    model.add(Conv2D(16, (5, 5)))
    model.add(Activation("relu"))
    model.add(MaxPooling2D((2,2)))
    model.add(Flatten())
    model.add(Dense(384))
    model.add(Activation("relu"))
    model.add(Dense(160))
    model.add(Activation("relu"))
    model.add(Dense(160))
    model.add(Activation("relu"))
    model.add(Dense(160))
    model.add(Activation("relu"))
    model.add(Dense(nclass))
    model.add(Activation("softmax"))
    model.compile(optimizer=SGD(0.01), loss="categorical_crossentropy",
              metrics=["accuracy"])

    return model

def positionWithMaxValueFromAVector(vector):
    valueToSearch=max(vector)
    i=0
    for x in vector:
        if(x==valueToSearch):
            return i            
        i=i+1
        
    return -1
