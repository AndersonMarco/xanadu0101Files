Uso de redes neurais para a identificação de frutas parecidas¶

Introdução¶

Este trabalho prope uma anlise sobre o uso de redes neurais para classificar frutas que so parecidas visualmente mas que so de diferentes tipos, criando assim um classificador de frutas. Para a realizao desta tarefa foi utilizado a base de dados Fruits 360, que uma base de dados que possui fotos de frutas, esta base esta disponvel em: https://www.kaggle.com/moltean/fruits

Este trabalho apresenta uma pequena descrio sobre as frutas utilizadas, uma anlise sobre os histogramas de cor para as fotos das frutas utilizadas, uma pequena descrio do tipo de rede neural utilizada para criar o classificador, cdigos utilizados para a criao do classificador e uma anlise sobre a eficincia que o classificador criado possui.

Os cdigos e artefatos utilizados para a realizao deste trabalho podem ser obtidos em: https://rebrand.ly/7sv33x

Elementos da base de dados utilizados para a análise¶

As frutas utilizadas para esta anlise so de trs diferentes tipos de tomate e para classificar de que tipo cada tomate foram utilizados dados extrados apenas de suas fotos. Um detalhe importante sobre a base de dados utilizada que ela pode ter at trs fotos de um mesmo tomate com cada foto sendo tirada de um ngulo diferente, com estes ngulos sendo fixos. As fotos utilizadas para esta anlise possuem 100 pixels de comprimento por 100 pixels de altura.

def computeHistogram(imgChannel):
    hist=np.zeros(256)
    for x in imgChannel:
        hist[x]=hist[x]+1.0

    return hist

def computeAvg(histsList):
    histAvg=np.zeros(256)
    for i in range(256):
        histAvg[i]=np.mean(histsList[:,i])

    return histAvg

def computeSTD(histsList):
    histSTD=np.zeros(256)
    for i in range(256):
        histSTD[i]=np.std(histsList[:,i])

    return histSTD

Rede neural utilizada¶

Para a criao e utilizao do classificador criado no so considerados o angulo da foto tirada e nem se um mesmo tomate possui varias fotos, apenas fornecido uma foto qualquer e o classificador infere o tipo de tomate que esta na foto.

A rede neural utilizada para criar o classificador do tipo convulacional, redes neurais deste tipo so parecidas com as as MLP (Multilayer Perceptron) possuindo tambm vrias camadas de neurnios com a diferena de que algumas destas camadas so do tipo convulacionais. Camadas convulacionais so camadas que criam filtros para os dados deste modo, as imagens passadas para uma rede neural deste tipo no precisam de um pr-processamento, como estes filtros so criados a medida que a rede neural treinada eles tentam ajustar-se para a forma que melhor atendam o classificador criado pela rede neural. A rede neural utiliza neste trabalho possui 4 camadas convulacionais perto da entrada e quatro camadas intermediarias do tipo MLP (ou camadas do tipo clssicas).

from keras.models import load_model

def createModel(height,width,channels,nclass):
    inputShape = (height, width, channels)

    model = Sequential()
    model.add(Conv2D(6, (5, 5), padding="same",input_shape=inputShape))
    model.add(Activation("relu"))
    model.add(MaxPooling2D((2,2)))
    model.add(Conv2D(16, (5, 5)))
    model.add(Activation("relu"))
    model.add(MaxPooling2D((2,2)))
    model.add(Flatten())
    model.add(Dense(384))
    model.add(Activation("relu"))
    model.add(Dense(160))
    model.add(Activation("relu"))
    model.add(Dense(160))
    model.add(Activation("relu"))
    model.add(Dense(160))
    model.add(Activation("relu"))
    model.add(Dense(nclass))
    model.add(Activation("softmax"))
    model.compile(optimizer=SGD(0.01), loss="categorical_crossentropy",
              metrics=["accuracy"])

    return model


config = tf.ConfigProto(allow_soft_placement=True, log_device_placement=False)

def positionWithMaxValueFromAVector(vector):
    valueToSearch=max(vector)
    i=0
    for x in vector:
        if(x==valueToSearch):
            return i            
        i=i+1
        
    return -1
